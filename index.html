Built-in rules

Quick reference for how this calculator works.

R
  • Upgrade: no plan payment, no access charge, no TWP. Device + tax + add-ons only.
  • Add a Line on existing Total account: no upfront plan payment. Device + access + TWP + tax + add-ons only.
  • New Number / Switch single-line: upfront months apply based on the promo row.
  • Free phone: tax is $0. Paid phones are taxed on the offer/device price, not SRP.
  • TWP: $8 per phone line unless it is an upgrade.
  • Access charge: $25 per line unless it is an upgrade.
  • Family plan: Total 5G = $55/$30/$25/$0. Total 5G+ = $65/$30/$25/$0.
